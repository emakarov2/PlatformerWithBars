 Health Display System - Demo Scene
   
   Чтобы протестировать систему:
   1. Откройте сцену DemoScene в папке Samples/Demo/
   2. Нажмите Play
   3. Используйте кнопки для нанесения урона и лечения
   
   Компоненты:
   - HealthText: текстовый индикатор
   - HealthBar: обычная полоса здоровья
   - SmoothHealthBar: плавная полоса здоровья
